# PONG

Pong game written in C, using raylib 4.5.0

This project is written for me to learn the basics of raylib.

`assets/button_*` are not used.

## Controls

- P1 (left)
    - w, s
- P2 (right)
    - arrow key up, down

Player controls P2 in one player mode

## Compiling

Compile Pong with

```sh
make
```

and run with

```sh
make run
```
